# Bank-Marketing-Customer-Response-Prediction

- Developed a machine learning system to predict customer response in bank marketing campaigns
- Analyzed the Bank Marketing dataset from UCI with demographic and financial features
- Performed data preprocessing and feature engineering (feature grouping, removal of data leakage variables)
- Addressed class imbalance using class weighting techniques
- Built and compared multiple models (Logistic Regression, Random Forest, KNN, SVM, XGBoost)
- Evaluated performance using cross-validation and metrics (Precision, Recall, F1-score)
